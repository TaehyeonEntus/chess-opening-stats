== Chess Opening Stats API

[[Player-Profile]]
=== 플레이어 존재 여부 확인
특정 체스 플랫폼(Chess.com, Lichess)에 해당 사용자 아이디를 가진 플레이어가 실제로 존재하는지 확인합니다.
존재할 경우 플레이어의 프로필 이미지 URL과 마지막 온라인 접속 시간을 반환합니다.

operation::get-profile[snippets='http-request,query-parameters,http-response,response-fields']

[[Task]]
=== 게임 분석 대기열 등록
플레이어의 최근 게임 데이터를 가져와 분석하도록 대기열에 등록합니다.
등록이 완료되면 200 OK를 반환하며, 이후 `/dashboard` 엔드포인트를 통해 분석 결과를 조회할 수 있습니다.

operation::enqueue-task[snippets='http-request,query-parameters,http-response']

[[Dashboard]]
=== 대시보드 조회 (Polling)
분석이 완료된 플레이어의 대시보드 데이터를 조회합니다.
분석이 진행 중이거나 등록되지 않은 경우 404 Not Found를 반환하므로, 클라이언트는 이 엔드포인트를 주기적으로 호출(Polling)해야 합니다.

operation::get-dashboard[snippets='http-request,query-parameters,http-response,response-fields']
