[[Security]]
=== Security & CSRF
본 API 서버는 CSRF(Cross-Site Request Forgery) 공격을 방지하기 위해 **CSRF 토큰**을 사용합니다.

* **대상 메서드**: `POST`, `PATCH`, `DELETE`, `PUT`
* **요청 헤더**: `X-CSRF-TOKEN`

[[Error-Response]]
=== Error Responses
API 호출 중 에러 발생 시 공통 응답 형식입니다.

[cols="1,2,3"]
|===
| 필드명 | 타입 | 설명

| `message`
| `String`
| 에러의 상세 사유 (프론트엔드에서 노출하거나 디버깅용으로 사용)
|===
